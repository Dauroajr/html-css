Principais formatações

Negrito (ou Destaque)

Nesta frase, temos um termo em negrito usando a tag < b >.

Nesta frase, temos um termo em destaque usando a tag < strong >.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag < i > (não semântico).

Nesta frase, temos um termo com ênfase usando a tag < em > (com semântica).

Texto Marcado

Podemos criar também um texto marcado com a tag < mark >.

E no outro parágrafo,temos outro texto marcado no final.

Texto Grande e texto Pequeno

Estamos criando um texto grande e um texto pequeno nesta frase.

Texto Excluído/Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não deve ser considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Existe também o texto sublinhado com a tag < U > (não semântico).

Texto Sobrescrito

Para inserir coisas como x3 + 3 (x ao cubo + 3).

Texto Subscrito

Para inserir coisas como H2O (fómula da água).
